# todo-app-python
To-Do List (Python)

A simple command-line To-Do List application built with Python.
It allows users to add, view, and remove tasks, with all data stored in a local tasks.txt file.

Features

View all tasks

Add new tasks

Remove tasks by number

Automatic saving and loading


Usage

Run the application with:

python todo.py

Requirements

Python 3.x
